
Running Cpp API examples
------------------------


#. 
   Running the Publisher example:

   .. code-block:: sh

           $ ./pub

#. 
   Running the Subscriber example:

   .. code-block:: sh

           $ ./sub

#. 
   Running the Server example:

   .. code-block:: sh

           $ ./server

#. 
   Running the Client example:

   .. code-block:: sh

           $ ./client

#. 
   Running the GetConfig & watchCallback example:

   .. code-block:: sh

           $ ./sample_app
